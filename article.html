This article demonstrates my attempts to NBconvert a Jupyter notebook into a distill.pub HTML page. The Jinja2 template named report.tpl can surely be improved upon.

NBconverting

With the file report.tpl in the same current directory as the notebook you want to convert, run 

jupyter nbconvert --to html --template report.tpl article.ipynb

Features

The title and abstract of this article are stored in this notebook's metadata. So too are the authors and their affiliations. Since distill.pub encourages authors and their affiliations to have associated URLs, the metadata that stores authors/affiliations is a list of length two lists where URLs are stored in the second element of the inner list.

This is a markdown cell that has a citation . It took a fairly ugly hack to get this to work. An outcome of this hack is that **markdown** does not work like it *should*, but the equivalent HTML tags will work. This goes the same for the Section headline above, which is coded with <h2> tags.

It is really the combination that citations and markdown that don't agree. For instancce, this cell, with tag only_md, is perfectly happy translating markdown into HTML.

Plots can have captions, specified via a key named caption in the cell's metadata.

x = np.linspace(-4, 4, 101)
y = normal.pdf(x)
bp.curve(x, y)
bp.labels(x="$z$", y="$f(z)$", size=18)

bp.save('normal.svg');
